# PRODUCT — WhatsApp Order Generator

## What it does

Generate a complete WhatsApp order message in seconds.

---

## Why it exists

Manual WhatsApp orders are messy:
- missing details
- repeated clarification
- inconsistent format

This tool fixes that.

---

## What you get

- structured order message
- ready to send
- consistent format

---

## Example

Input:
- Name: Ali
- Product: Sambal Pedas
- Qty: 2

Output:

````

ORDER BARU

Nama: Ali

Item Pesanan

Item 1
Produk: Sambal Pedas
Kuantiti: 2
Catatan: -

```

---

## Who it is for

- small sellers
- WhatsApp-based businesses
- dropship agents

---

## What it is NOT

- not an online store
- not order management
- not payment system

---

## Value

- faster order intake
- fewer mistakes
- cleaner communication
